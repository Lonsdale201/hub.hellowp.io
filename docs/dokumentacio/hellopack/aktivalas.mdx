# <FontAwesomeIcon icon="fa-duotone fa-command text-primary" /> HelloPack csomagkezelő aktiválása

Miután telepítetted és aktiváltad a HelloPack csomagkezelőt, a következő lépés, hogy aktiváld a csomagkezelőt a weboldaladon.

### 1. Kattints az adminban a **HelloPack** feliratra

Kattints a WordPress admin oldalsó menüjében a **HelloPack** feliratra, hogy eljusson a csomagkezelő felületére.

### 2. HelloPack aktiválása felületet

1. Másold ki az API kulcsodat a HelloWP.io oldal konzoljából: https://hellowp.io/hu/
   helloconsole/hellopack-kozpont/

2. Illeszd be az API kulcsodat a HelloPack aktiválása felületen az **API kulcs** mezőbe.

3. Kattints a **HelloPack aktiválása** gombra.

<BrowserWindow url="https://peldaweboldal.hu">
  <img src="/img/hellopack-aktivalasa.png" alt="Kép leíró szövege"></img>
</BrowserWindow>

### 3. HelloPack aktív

Amennyiben minden rendben és a HelloPack aktiválása sikeres volt, akkor a HelloPack aktivációs felületén ezt kell látnod:

<BrowserWindow url="https://peldaweboldal.hu">
  <img src="/img/hellopack-minden-ok.png" alt="Kép leíró szövege"></img>
</BrowserWindow>

### <FontAwesomeIcon icon="fa-duotone fa-bug text-danger" /> Hibára futottál?

Amennyiben valamilyen hiba történt a HelloPack aktiválása során, akkor nézd át a [HelloPack hibaelhárítási útmutatót](/docs/dokumentacio/hellopack/hibaelharitas).
